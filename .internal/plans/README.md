# Implementation Plans

This directory contains implementation plans for APG patterns.

## Directory Structure

```
plans/
├── README.md          # This file
├── completed/         # Completed implementation plans (archived)
└── (new plans here)   # Active plans
```

## Creating New Plans

For new patterns or features, create a new `.md` file in this directory using the pattern:

```
{feature-name}-implementation-plan.md
```
