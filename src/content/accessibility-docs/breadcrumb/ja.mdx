### WAI-ARIA ロール

| ロール       | 対象要素     | 説明                                                                      |
| ------------ | ------------ | ------------------------------------------------------------------------- |
| `navigation` | `<nav>` 要素 | 支援技術にナビゲーションランドマークを提供します（`<nav>`の暗黙のロール） |

### WAI-ARIA プロパティ

#### `aria-label`

スクリーンリーダー向けにナビゲーションランドマークにラベルを付けます

<KeyValueList>
  <KeyValueListItem term="値">`Breadcrumb`（またはローカライズされた値）</KeyValueListItem>
  <KeyValueListItem term="必須">はい</KeyValueListItem>
</KeyValueList>

#### `aria-current`

パンくずリスト内の現在のページを識別します

<KeyValueList>
  <KeyValueListItem term="値">`page`</KeyValueListItem>
  <KeyValueListItem term="必須">はい（最後のアイテム）</KeyValueListItem>
</KeyValueList>

### WAI-ARIA ステート

#### `aria-current`

<KeyValueList>
  <KeyValueListItem term="対象要素">
    パンくずリストの最後のアイテム（現在のページ）
  </KeyValueListItem>
  <KeyValueListItem term="値">`page`</KeyValueListItem>
  <KeyValueListItem term="必須">はい</KeyValueListItem>
  <KeyValueListItem term="変更トリガー">
    パンくずリストナビゲーション内の現在のページ位置を示します。
  </KeyValueListItem>
</KeyValueList>

### キーボードサポート

| キー             | アクション                                       |
| ---------------- | ------------------------------------------------ |
| <kbd>Tab</kbd>   | パンくずリストのリンク間でフォーカスを移動します |
| <kbd>Enter</kbd> | フォーカスされたリンクをアクティブ化します       |

- Breadcrumbはネイティブの`<a>`要素の動作をキーボードインタラクションに使用します。追加のキーボードハンドラーは不要です。
