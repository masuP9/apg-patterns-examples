### WAI-ARIA ロール

| ロール | 対象要素 | 説明 |
|------|------|------|
| `navigation` | `<nav>` 要素 | 支援技術にナビゲーションランドマークを提供します（`<nav>`の暗黙のロール） |

### WAI-ARIA プロパティ

#### `aria-label`

スクリーンリーダー向けにナビゲーションランドマークにラベルを付けます

| | |
|---|---|
| **値** | "Breadcrumb"（またはローカライズされた値） |
| **必須** | はい |

#### `aria-current`

パンくずリスト内の現在のページを識別します

| | |
|---|---|
| **値** | "page" |
| **必須** | はい（最後のアイテム） |

### WAI-ARIA ステート

#### `aria-current`

| | |
|---|---|
| **対象要素** | パンくずリストの最後のアイテム（現在のページ） |
| **値** | "page" |
| **必須** | はい |
| **変更トリガー** | パンくずリストナビゲーション内の現在のページ位置を示します。 |

### キーボードサポート

| キー | アクション |
|------|------|
| <kbd>Tab</kbd> | パンくずリストのリンク間でフォーカスを移動します |
| <kbd>Enter</kbd> | フォーカスされたリンクをアクティブ化します |

> Breadcrumbはネイティブの`<a>`要素の動作をキーボードインタラクションに使用します。追加のキーボードハンドラーは不要です。
