### WAI-ARIA Roles

| Role           | Target Element     | Description                                                        |
| -------------- | ------------------ | ------------------------------------------------------------------ |
| `grid`         | Container          | Identifies the element as a grid. The grid contains rows of cells. |
| `row`          | Each row           | Identifies a row of cells                                          |
| `gridcell`     | Each cell          | Identifies an interactive cell in the grid                         |
| `rowheader`    | Row header cell    | Identifies a cell as a header for its row                          |
| `columnheader` | Column header cell | Identifies a cell as a header for its column                       |

### Keyboard Support

| Key                      | Action                                                              |
| ------------------------ | ------------------------------------------------------------------- |
| <kbd>ArrowRight</kbd>    | Move focus one cell to the right. Wraps to next row if at end.      |
| <kbd>ArrowLeft</kbd>     | Move focus one cell to the left. Wraps to previous row if at start. |
| <kbd>ArrowDown</kbd>     | Move focus one cell down.                                           |
| <kbd>ArrowUp</kbd>       | Move focus one cell up.                                             |
| <kbd>Home</kbd>          | Move focus to the first cell in the row.                            |
| <kbd>End</kbd>           | Move focus to the last cell in the row.                             |
| <kbd>Ctrl + Home</kbd>   | Move focus to the first cell in the grid.                           |
| <kbd>Ctrl + End</kbd>    | Move focus to the last cell in the grid.                            |
| <kbd>Page Down</kbd>     | Move focus down by a page (implementation-defined).                 |
| <kbd>Page Up</kbd>       | Move focus up by a page (implementation-defined).                   |
| <kbd>Space / Enter</kbd> | Activate the cell (e.g., edit, select).                             |
| <kbd>Escape</kbd>        | Cancel edit mode or deselect.                                       |

- Use role="grid" only when the table is interactive. For static data, use native `<table>` elements.
- Roving tabindex is recommended for efficient keyboard navigation.
- Consider providing a visible focus indicator for the focused cell.

### Focus Management

| Event                        | Behavior                                              |
| ---------------------------- | ----------------------------------------------------- |
| Grid                         | `tabindex="0"` on container or first focusable cell   |
| Focused cell                 | `tabindex="0"`                                        |
| Other cells                  | `tabindex="-1"`                                       |
| Interactive content in cells | Focus moves into cell content on Enter, out on Escape |
