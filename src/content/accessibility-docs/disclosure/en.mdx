### WAI-ARIA Roles

| Role     | Target Element  | Description                                                               |
| -------- | --------------- | ------------------------------------------------------------------------- |
| `button` | Trigger element | Interactive element that toggles panel visibility (use native `<button>`) |

### WAI-ARIA Properties

#### `aria-controls`

Associates the button with the panel it controls

<KeyValueList>
  <KeyValueListItem term="Values">ID reference to panel</KeyValueListItem>
  <KeyValueListItem term="Required">Yes</KeyValueListItem>
</KeyValueList>

#### `aria-hidden`

Hides panel from assistive technology when collapsed

<KeyValueList>
  <KeyValueListItem term="Values">true | false</KeyValueListItem>
  <KeyValueListItem term="Required">No</KeyValueListItem>
</KeyValueList>

### WAI-ARIA States

#### `aria-expanded`

<KeyValueList>
  <KeyValueListItem term="Target Element">button element</KeyValueListItem>
  <KeyValueListItem term="Values">true | false</KeyValueListItem>
  <KeyValueListItem term="Required">Yes</KeyValueListItem>
  <KeyValueListItem term="Change Trigger">Click, Enter, Space</KeyValueListItem>
</KeyValueList>

### Keyboard Support

| Key                      | Action                                        |
| ------------------------ | --------------------------------------------- |
| <kbd>Tab</kbd>           | Move focus to the disclosure button           |
| <kbd>Space / Enter</kbd> | Toggle the visibility of the disclosure panel |

- Disclosure uses native `<button>` element behavior for keyboard interaction. No additional keyboard handlers are required.
