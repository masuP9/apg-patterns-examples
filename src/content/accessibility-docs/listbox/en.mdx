### WAI-ARIA Roles

| Role      | Target Element     | Description                                        |
| --------- | ------------------ | -------------------------------------------------- |
| `listbox` | Container (`<ul>`) | Widget for selecting one or more items from a list |
| `option`  | Each item (`<li>`) | Selectable option within the listbox               |

### Keyboard Support

#### Common Navigation

| Key                                 | Action                                                    |
| ----------------------------------- | --------------------------------------------------------- |
| <kbd>Down Arrow / Up Arrow</kbd>    | Move focus (vertical orientation)                         |
| <kbd>Right Arrow / Left Arrow</kbd> | Move focus (horizontal orientation)                       |
| <kbd>Home</kbd>                     | Move focus to first option                                |
| <kbd>End</kbd>                      | Move focus to last option                                 |
| <kbd>Type character</kbd>           | Type-ahead: focus option starting with typed character(s) |

#### Single-Select (Selection Follows Focus)

| Key                      | Action                                  |
| ------------------------ | --------------------------------------- |
| <kbd>Arrow keys</kbd>    | Move focus and selection simultaneously |
| <kbd>Space / Enter</kbd> | Confirm current selection               |

#### Multi-Select

| Key                      | Action                                |
| ------------------------ | ------------------------------------- |
| <kbd>Arrow keys</kbd>    | Move focus only (selection unchanged) |
| <kbd>Space</kbd>         | Toggle selection of focused option    |
| <kbd>Shift + Arrow</kbd> | Move focus and extend selection range |
| <kbd>Shift + Home</kbd>  | Select from anchor to first option    |
| <kbd>Shift + End</kbd>   | Select from anchor to last option     |
| <kbd>Ctrl + A</kbd>      | Select all options                    |

- **Single-select:** Selection follows focus (arrow keys change selection)
- **Multi-select:** Focus and selection are independent (Space toggles selection)

### Focus Management

| Event               | Behavior                                                       |
| ------------------- | -------------------------------------------------------------- |
| Focused option      | Only one option has `tabindex="0"` at a time (Roving Tabindex) |
| Non-focused options | Other options have `tabindex="-1"`                             |
| Arrow navigation    | Arrow keys move focus between options                          |
| Disabled options    | Disabled options are skipped during navigation                 |
| Edge behavior       | Focus does not wrap (stops at edges)                           |
