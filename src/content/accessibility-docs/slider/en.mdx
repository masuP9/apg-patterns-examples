### WAI-ARIA Roles

| Role | Target Element | Description |
|------|------|------|
| `slider` | Thumb element | Identifies the element as a slider that allows users to select a value from within a range |

### WAI-ARIA Properties

#### `aria-valuenow`

Updated dynamically when value changes via keyboard or pointer

| | |
|---|---|
| **Values** | Number |
| **Required** | Yes |

#### `aria-valuemin`

Default: 0

| | |
|---|---|
| **Values** | Number |
| **Required** | Yes |

#### `aria-valuemax`

Default: 100

| | |
|---|---|
| **Values** | Number |
| **Required** | Yes |

#### `aria-valuetext`

Example: "50%", "Medium", "3 of 5 stars"

| | |
|---|---|
| **Values** | String |
| **Required** | No (recommended when value needs context) |

#### `aria-orientation`

Default: horizontal (implicit). Only set for vertical sliders.

| | |
|---|---|
| **Values** | "horizontal" \| "vertical" |
| **Required** | No |

#### `aria-disabled`

Set only when disabled

| | |
|---|---|
| **Values** | true \| undefined |
| **Required** | No |

### Keyboard Support

| Key | Action |
|------|------|
| <kbd>Right Arrow</kbd> | Increases the value by one step |
| <kbd>Up Arrow</kbd> | Increases the value by one step |
| <kbd>Left Arrow</kbd> | Decreases the value by one step |
| <kbd>Down Arrow</kbd> | Decreases the value by one step |
| <kbd>Home</kbd> | Sets the slider to its minimum value |
| <kbd>End</kbd> | Sets the slider to its maximum value |
| <kbd>Page Up</kbd> | Increases the value by a large step (default: step * 10) |
| <kbd>Page Down</kbd> | Decreases the value by a large step (default: step * 10) |

> Sliders must have an accessible name via visible label (recommended), aria-label, or aria-labelledby.

> This implementation follows WCAG guidelines for accessible visual design including focus indicators, visual fill, and hover states.

> Pointer interaction: Click on track moves thumb immediately; drag allows continuous adjustment.

> Forced colors mode uses system colors for Windows High Contrast Mode accessibility.

### Implementation Notes

```
Structure:
<div class="slider-container">
  <label id="label">Volume</label>
  <div class="slider-track">
    <div class="slider-fill" />
    <div
      role="slider"
      tabindex="0"
      aria-valuenow="50"
      aria-valuemin="0"
      aria-valuemax="100"
      aria-labelledby="label"
      class="slider-thumb"
    />
  </div>
</div>

Visual Layout:
┌─────────────────────────────────────┐
│ Volume                               │
│ ├─────────●─────────────────────────┤
│ 0        50                     100  │
└─────────────────────────────────────┘

Keyboard Navigation:
← / ↓  = Decrease by step
→ / ↑  = Increase by step
Home   = Set to min
End    = Set to max
PgUp   = Increase by large step
PgDn   = Decrease by large step
```
